# About

This project is trying to "sha-kyo (写経)" a book "Google App Engine for Java実践ガイド (Author : 小川信一)" with golang.

# prepare

1. install go
2. set $GOPATH
    * This project use `direnv`.
3. Donwload appengine SDK

# start dev-server

`$ goapp serve .`

# deploy to appengine

`$ goapp deploy .`
